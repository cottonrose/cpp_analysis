打字 + 全选复制粘贴到 n 字的最少步数

📄 题目描述

A 打字
字数 +1
B 全选 + 复制 + 粘贴
字数 ×2
初始有 1 个字，每次可选操作 A 或 B，恰好达到 n 个字，求最少操作次数。
12
📋 动态规划五步法
① 划分阶段
按照当前字数 i 划分阶段，1 ≤ i ≤ n
② 状态表示
dp[i] = 达到 i 个字的最少操作次数
③ 状态转移方程
dp[i] = min(dp[i−1] + 1,
i%2=0 ? dp[i/2] + 1 : ∞)
④ 边界条件
dp[1] = 0（已有 1 个字，无需操作）
⑤ 答案
dp[n] = ?
💻 AC 参考代码 
// DP  O(n)  n ≤ 10⁶
#include <bits/stdc++.h>
using namespace std;

int main() {
  int n; cin >> n;
  int dp[n+1];
  dp[1] = 0;                              // ④ 边界
  for (int i = 2; i <= n; i++) {
    dp[i] = dp[i-1] + 1;                  // ③ +1 操作
    if (i % 2 == 0)
      dp[i] = min(dp[i], dp[i/2] + 1); // ③ ×2 操作
  }
  cout << dp[n] << '\n';                 // ⑤ 答案
  return 0;
}

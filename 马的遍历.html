📋 题目描述

有一个 n × m 的棋盘，在某个点 (x, y) 上有一个马。
要求计算出马到达棋盘上任意一个点最少要走几步（马走"日"字，8 个方向）。
不能到达则输出 -1。

数据范围：1 ≤ x ≤ n ≤ 400，1 ≤ y ≤ m ≤ 400（演示中限制 ≤ 15 以便显示）

💻 参考代码 (C++)

// P1443 马的遍历 — BFS 解法 #include <bits/stdc++.h> using namespace std; const int N = 405; int n, m, x, y; int dist[N][N]; int dx[] = {1,2,2,1,-1,-2,-2,-1}; int dy[] = {2,1,-1,-2,-2,-1,1,2}; void bfs() { memset(dist, -1, sizeof(dist)); queue<pair<int,int>> q; dist[x][y] = 0; q.push({x, y}); while (!q.empty()) { auto [cx, cy] = q.front(); q.pop(); for (int i = 0; i < 8; i++) { int nx = cx + dx[i]; int ny = cy + dy[i]; if (nx >= 1 && nx <= n && ny >= 1 && ny <= m && dist[nx][ny] == -1) { dist[nx][ny] = dist[cx][cy] + 1; q.push({nx, ny}); } } } } int main() { cin >> n >> m >> x >> y; bfs(); for (int i = 1; i <= n; i++) { for (int j = 1; j <= m; j++) cout << dist[i][j] << " "; cout << endl; } }

🧠 解题思路

① 把棋盘上每个位置都当作一个状态。
② 初始状态为起点 (x, y)。
③ 若两个状态可通过马走一步到达，则两状态互相可达。
④ 题目要求最少步数 → 等价于求最短路。
⑤ 边权均为 1 的最短路 → 使用广度优先搜索 (BFS)。

💡 BFS 的逐层扩展特性保证了第一次到达某个格子时的步数就是最短步数。 马走"日"字共 8 个方向：(±1, ±2) 和 (±2, ±1) 的组合。

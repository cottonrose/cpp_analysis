🛗 P1135 奇怪的电梯

BFS 过程演示

📋 题目描述

大楼有 N 层，第 i 层有一个数字 Ki。 电梯只有四个按钮：开、关、上、下。
按"上"则上升 Ki 层，按"下"则下降 Ki 层（超出 1~N 范围则按钮失灵）。
求从 A 楼到 B 楼最少按键次数，无法到达输出 -1。

样例：N=5, A=1, B=5, K=[3,3,1,2,5] → 答案 3（1→4→2→5）

💻 参考代码 (C++)

// P1135 奇怪的电梯 — BFS #include <bits/stdc++.h> using namespace std; const int N = 205; int n, a, b, k[N]; int dis[N]; // dis[i] = 从起点到i的最少步数 int bfs() { memset(dis, -1, sizeof(dis)); queue<int> q; dis[a] = 0; q.push(a); while (!q.empty()) { int u = q.front(); q.pop(); if (u == b) return dis[u]; // 尝试上楼 int up = u + k[u]; if (up <= n && dis[up] == -1) { dis[up] = dis[u] + 1; q.push(up); } // 尝试下楼 int dn = u - k[u]; if (dn >= 1 && dis[dn] == -1) { dis[dn] = dis[u] + 1; q.push(dn); } } return -1; } int main() { cin >> n >> a >> b; for (int i = 1; i <= n; i++) cin >> k[i]; cout << bfs() << endl; }

🧠 解题思路

① 把当前楼层编号看作一个状态。
② 若楼层 i 可通过一次按钮到达楼层 j，则 i → j。
③ 题目要求最少按钮次数 → 等价于从 A 到 B 的最短路径。
④ 每层恰好有两个方向（上 Ki 层 / 下 Ki 层），边权均为 1。
⑤ 边权为 1 的最短路 → BFS。

💡 与 P1443「马的遍历」类似，但这里是一维 BFS：每个节点只有 2 个扩展方向而非 8 个。 BFS 第一次到达某楼层时的步数就是最短步数。

💻 参考代码 (C++)

// 九九归一 — 模9性质解法 #include <bits/stdc++.h> using namespace std; int main() { ios::sync_with_stdio(false); cin.tie(nullptr); int T; cin >> T; while (T--) { string s; cin >> s; // 利用模9性质：只需计算各位数字之和模9 int sum = 0; for (char ch : s) { sum += ch - '0'; } // 数字根公式 if (sum % 9 == 0) { cout << 9 << '\n'; } else { cout << sum % 9 << '\n'; } } return 0; }

🧠 解题思路

① 核心性质：n ≡ 各位数字之和 (mod 9)。
② 反复求各位数字之和，本质上是在不断求 n 模 9 的值。
③ 最终的一位数就是 数字根 (Digital Root)：
  若 n = 0，结果为 0；
  若 n % 9 = 0，结果为 9；
  否则结果为 n % 9。
④ 等价公式：dr(n) = 1 + (n - 1) % 9（n > 0 时成立）。

💡 由于输入可能很大（长度 10⁵），不能使用 int/long long 直接存，需用字符串读入后遍历求和。

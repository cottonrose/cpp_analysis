逆元之和 — 线性逆元递推演示

数论 · 乘法逆元

📋 题目描述

给定素数 $p$ 和正整数 $n$（$n < p$），计算 $S = \sum_{i=1}^{n} i^{-1} \pmod p$。

使用线性递推公式 $\text{inv}[i] = p - \lfloor p/i \rfloor \times \text{inv}[p \bmod i] \bmod p$ 在 $O(n)$ 时间内求出所有逆元。该公式不要求 $p$ 为素数，只要 $p \bmod i$ 的逆元存在即可。

速度： 800ms 💡 → ← Space 💡 n 较小时体验最佳（建议 n ≤ 30）

// 线性逆元模板 — 逆元之和 #include <bits/stdc++.h> using namespace std; using ll = long long; const int MAXN = 5e6 + 5; ll inv[MAXN]; int main() { ios::sync_with_stdio(false); cin.tie(nullptr); int n; ll p; cin >> n >> p; inv[1] = 1; ll sum = 1; for (int i = 2; i <= n; ++i) { inv[i] = p - (p / i) * inv[p % i] % p; sum = (sum + inv[i]) % p; } cout << sum << '\n'; return 0; }

🧠 解题思路

核心公式：$\text{inv}[i] = p - \lfloor p/i \rfloor \times \text{inv}[p \bmod i] \bmod p$

推导：设 $p = k \cdot i + r$，其中 $k = \lfloor p/i \rfloor$, $r = p \bmod i$。在模 $p$ 意义下 $k \cdot i + r \equiv 0$，两边同乘 $i^{-1} \cdot r^{-1}$ 得 $k \cdot r^{-1} + i^{-1} \equiv 0$，移项即得公式。

三种方法对比：扩展欧几里得 $O(n \log p)$ → 50 分（$n$ 较小时可用）；费马小定理 + 快速幂 $O(n \log p)$ → 80 分（⚠️ 仅在 $p$ 为素数时正确）；线性逆元 $O(n)$ → 100 分（不依赖 $p$ 是否为素数）。

💻 参考代码 (C++)

// 和式求和 — O(n) 公式解法 #include <bits/stdc++.h> using namespace std; const long long MOD = 1e9 + 7; int main() { ios::sync_with_stdio(false); cin.tie(nullptr); int T; cin >> T; while (T--) { int n; cin >> n; long long sum = 0; for (int i = 0; i < n; i++) { int x; cin >> x; sum = (sum + x) % MOD; } long long ans = ((n - 1) % MOD) * sum % MOD; cout << ans << '\n'; } return 0; }

🧠 解题思路

① 直接双重循环 $O(n^2)$ 在大数据下会超时。
② 考察每个 $a_k$ 在总和中出现的次数：  作为 $a_i$（$i=k$）出现 $(n-k)$ 次，  作为 $a_j$（$j=k$）出现 $(k-1)$ 次。
③ 每个 $a_k$ 共出现 $(n-k)+(k-1) = n-1$ 次。
④ $S = (n-1) \times \sum_{i=1}^{n} a_i$，$O(n)$ 一次遍历即可。

P1002 过河卒

NOIP 2002 普及组 · 二维 DP · 马拦过河卒

📄 题目描述

🐾 卒
从 (0,0) 走到 (n,m)
➡️ 移动
只能 向右 或 向下 走一步
🐴 马的控制点
马所在位置 + 8个 日 字跳点 不可经过
🎯 目标
计算从起点到终点的 路径条数
, ,

🗺️ 棋盘 设置后点「开始」

未计算 已计算 马的控制点 起点 终点

📝 说明

设置 B 点坐标和马的位置，点击「开始」。
到达终点的路径条数
0
f[0,0]
📋 动态规划五步法
① 划分阶段
按坐标 (i,j) 划分，0 ≤ i ≤ n，0 ≤ j ≤ m
② 状态表示
f[i][j] = 从 (0,0) 到 (i,j) 的路径条数
③ 状态转移方程
📌 马的控制点判定（check(i,j)）：
|i−mx|==1 && |j−my|==2 → 日字跳
|i−mx|==2 && |j−my|==1 → 日字跳
i==mx && j==my → 马本身
📌 若 check(i,j)=1：f[i][j] = 0（不可达）
📌 否则（i≥1, j≥1）：
f[i][j] = f[i−1][j] + f[i][j−1]
④ 边界条件
f[0][0] = 1（起点算 1 条路径）
第 0 列：i=0..n 顺序初始化，遇控制点 break，后续为 0
第 0 行：j=0..m 顺序初始化，遇控制点 break，后续为 0
⑤ 答案
f[n][m] = ?
💻 AC 参考代码 
// P1002 过河卒  O(nm)  n,m ≤ 20
#include <bits/stdc++.h>
using namespace std;

int n, m, mx, my;
long long f[25][25];

int check(int a, int b) {
    // 日字跳：马一步能跳到 (a,b) 则被控制
    if (abs(a - mx) == 1 && abs(b - my) == 2
    ||  abs(a - mx) == 2 && abs(b - my) == 1)
        return 1;
    // 马本身的位置
    if (a == mx && b == my)
        return 1;
    return 0;
}

int main() {
    cin >> n >> m >> mx >> my;

    // ④ 边界：第 0 列 — 遇控制点则 break，后续为 0
    for (int i = 0; i <= n; ++i)
        if (check(i, 0)) break;
        else f[i][0] = 1;

    // ④ 边界：第 0 行 — 遇控制点则 break，后续为 0
    for (int j = 0; j <= m; ++j)
        if (check(0, j)) break;
        else f[0][j] = 1;

    // ③ 状态转移：填表 O(nm)
    for (int i = 1; i <= n; ++i)
        for (int j = 1; j <= m; ++j)
            if (check(i, j))
                f[i][j] = 0;          // 马的控制点不可达
            else
                f[i][j] = f[i - 1][j] + f[i][j - 1];

    cout << f[n][m];                    // ⑤ 答案
    return 0;
}

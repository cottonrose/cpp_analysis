题目（小朋友版 🧸）

有两排小朋友，分别叫 X 队和 Y 队。请你从两排中选出最多的小朋友，要求：

  • 选出来的人在原来队伍里的顺序不能变（不能插队）
  • 选出来的人同时出现在 X 队和 Y 队里

问最多能选出几个人？这个数就叫 LCS 长度。

给小朋友的总结 📖

问题：两队小朋友（X 队和 Y 队），选出最多的相同的人，顺序不能乱。

方法：画一张表格，一行一行填数字。

  • f[i][j] = 只看 X 队前 i 个人 和 Y 队前 j 个人，最多能选几个
  • 看两个队伍最后一个人：
    • ✅ 一样 → 这个人一定能选上！f[i][j] = f[i-1][j-1] + 1
    • ❌ 不一样 → 选一个更长的方案：f[i][j] = max(f[i-1][j], f[i][j-1])
  • 一直填到右下角 f[n][m] 就是答案啦！

💡 边界：如果某个队一个小朋友都没有（空队），结果就是 0。

参考代码 (C++)

#include <iostream> #include <cstring> #include <algorithm> using namespace std; const int N = 1005; int n, m; int f[N][N]; char x[N], y[N]; int main() { cin >> (x + 1) >> (y + 1); n = strlen(x + 1); m = strlen(y + 1); for (int i = 1; i <= n; i++) { for (int j = 1; j <= m; j++) { if (x[i] == y[j]) { f[i][j] = f[i-1][j-1] + 1; } else { f[i][j] = max(f[i-1][j], f[i][j-1]); } } } cout << f[n][m] << endl; return 0; }

最长公共子序列 (Longest Common Subsequence)

动态规划 · O(n×m)

题目描述

给定两个序列 X 和 Y，求 X 和 Y 的最长公共子序列（LCS）的长度。

子序列：在原序列中删去若干元素（可以不删）后得到的序列，不要求连续，但元素间相对顺序不能改变。

解题思路

核心思想：把原问题拆成更小的子问题。

要求 X[1..i] 和 Y[1..j] 的 LCS，观察最后一个字符：

  • 如果 X[i] == Y[j]，说明这个字符可以成为 LCS 的一部分 → 长度 = X[1..i-1] 和 Y[1..j-1] 的 LCS 长度 + 1（即 f[i-1][j-1] + 1）
  • 如果 X[i] != Y[j]，则 X[i] 和 Y[j] 至少有一个不在 LCS 中 → 答案在 f[i-1][j] 和 f[i][j-1] 中取较大者

边界条件：空串的 LCS = 0，即 f[0][j] = f[i][0] = 0

最终答案：f[n][m]

时间复杂度 O(n×m)，空间复杂度 O(n×m)。

参考代码 (C++)

#include <iostream> #include <cstring> #include <algorithm> using namespace std; const int N = 1005; int n, m; int f[N][N]; char x[N], y[N]; int main() { cin >> (x + 1) >> (y + 1); n = strlen(x + 1); m = strlen(y + 1); for (int i = 1; i <= n; i++) { for (int j = 1; j <= m; j++) { if (x[i] == y[j]) { f[i][j] = f[i-1][j-1] + 1; } else { f[i][j] = max(f[i-1][j], f[i][j-1]); } } } cout << f[n][m] << endl; return 0; }

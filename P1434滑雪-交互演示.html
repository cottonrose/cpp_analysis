📋 动态规划五步法（排序 DP）
① 划分阶段
按高度升序处理每个格子。排序保证处理当前格时，更低相邻格子 f 值已算好。
② 状态表示
f[i][j] = 从 (i,j) 出发的最长滑雪路径长度（含自身）
③ 状态转移
遍历四个方向：更低 → f[x][y] = max(f[x][y], f[nx][ny])；完后 ++
记录 nxt[x][y] 指向最优下一格，用于还原路径
④ 边界
f 初始 0。局部最低点：f = 1
⑤ 答案
ans = max(f[i][j])。样例答案：25
💻 DP 排序版参考代码 
// P1434 [SHOI2002] 滑雪 — DP 排序版  O(nm log nm)
#include <bits/stdc++.h>
using namespace std;

const int N = 105;
int n, m, a[N][N], f[N][N], tot, ans;
int dx[4] = {0,0,1,-1};
int dy[4] = {1,-1,0,0};

struct node {
    int h, x, y;
    bool operator<(const node &x)const { return h < x.h; }
} b[N * N];

int main() {
    cin >> n >> m;
    for (int i = 1; i <= n; ++i)
        for (int j = 1; j <= m; ++j) {
            cin >> a[i][j];
            b[++tot] = {a[i][j], i, j};
        }
    sort(b + 1, b + 1 + tot);
    for (int i = 1; i <= tot; ++i) {
        int x = b[i].x, y = b[i].y;
        for (int j = 0; j <= 3; ++j) {
            int _x = x + dx[j], _y = y + dy[j];
            if (a[_x][_y] < a[x][y])
                f[x][y] = max(f[x][y], f[_x][_y]);
        }
        f[x][y]++;
        ans = max(ans, f[x][y]);
    }
    cout << ans;
    return 0;
}

💡 为什么需要优化？

方法思路时间复杂度特点
❌ 朴素 DFS 对每个起点递归搜索全部可能路径 O(4nm) 大量重复计算，同一个格子被反复访问
📊 排序 DP 按高度升序迭代处理 O(nm log nm) 需要排序，必须处理全部格子
⚡ 记忆化 DFS 递归 + 缓存，算过的不再重复 O(nm) 无需排序，只算可达的格子，代码最简洁

核心优化：f[x][y] 一旦算好就存下来，下次直接 return f[x][y]。这样每个格子只递归一次，指数级降为线性。

⚡ DFS 递归顺序

单元格标记：①递归调用顺序（先序） / ②缓存完成顺序（后序）

📋 动态规划五步法（记忆化搜索）
① 划分阶段
按 DFS 后序（先递归更低邻居，再返回自身）。每个格子被递归一次。
② 状态表示
f[x][y] = 从 (x,y) 出发的最长滑雪路径长度。-1 表示未计算。
③ 状态转移
递归遍历四个方向中更低者：
f[x][y] = max(f[x][y], dfs(nx,ny))
递归返回后 ++f[x][y]，存入缓存
④ 边界
f 初始 -1（表示未算）。
f[x][y] != -1 → 直接 return（缓存命中）
无更低邻居 → f = 0 → ++ → 1
⑤ 答案
对每个格子执行 ans = max(ans, dfs(i,j))，最终 ans 即为最长路径长度
💪 记忆化搜索优化代码 
// P1434 [SHOI2002] 滑雪 — 记忆化搜索  O(nm)
/* 优化思路：
 * 朴素 DFS 会反复计算同一个格子，导致指数级复杂度。
 * 用 f[x][y] 缓存计算结果，每个格子只算一次，
 * 时间复杂度从 O(4^nm) 降到 O(nm)，且无需排序。
 */
#include <bits/stdc++.h>
using namespace std;

const int N = 105;
int n, m, a[N][N], f[N][N], ans;
int dx[4] = {0,0,1,-1};
int dy[4] = {1,-1,0,0};

int dfs(int x, int y) {
    if (f[x][y] != -1)              // ③ 缓存命中
        return f[x][y];

    f[x][y] = 0;                       // ④ 初始 0
    for (int i = 0; i < 4; ++i) {
        int _x = x + dx[i], _y = y + dy[i];
        if (_x >= 1 && _x <= n && _y >= 1 && _y <= m
            && a[_x][_y] < a[x][y])
            f[x][y] = max(f[x][y], dfs(_x, _y)); // ③ 递归更低邻居
    }
    return ++f[x][y];                   // ③ +1 后返回并缓存
}

int main() {
    cin >> n >> m;
    for (int i = 1; i <= n; ++i)
        for (int j = 1; j <= m; ++j)
            cin >> a[i][j];

    memset(f, -1, sizeof(f));           // ④ -1 表示未计算

    for (int i = 1; i <= n; ++i)
        for (int j = 1; j <= m; ++j)
            ans = max(ans, dfs(i, j));          // ⑤ 取最大值

    cout << ans;
    return 0;
}

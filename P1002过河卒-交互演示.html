P1002 过河卒

NOIP 2002 普及组 · 二维 DP · 马拦过河卒

📄 题目描述

🐾 卒
从 (0,0) 走到 (n,m)
➡️ 移动
只能 向右 或 向下 走一步
🐴 马的控制点
马所在位置 + 8个 日 字跳点 不可经过
🎯 目标
计算从起点到终点的 路径条数
, ,

🗺️ 棋盘 设置后点「开始」

未计算 已计算 马的控制点 起点 终点 ← 当前

📝 说明

设置 B 点坐标和马的位置，点击「开始」。
到达终点的路径条数
0
dp[0,0]
📋 动态规划五步法
① 划分阶段
按坐标 (i,j) 划分，0 ≤ i ≤ n，0 ≤ j ≤ m
② 状态表示
dp[i][j] = 从 (0,0) 到 (i,j) 的路径条数
③ 状态转移方程
📌 一般情况：上 + 左
dp[i][j] = dp[i−1][j] + dp[i][j−1]
📌 控制点：dp[i][j] = 0
📌 第 0 行/列：只从一个方向来
④ 边界条件
dp[0][0] = 1（起点本身算 1 条路径）
⑤ 答案
dp[n][m] = ?
💻 AC 参考代码 
// P1002 过河卒  O(nm)  n,m ≤ 20
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

const int dx[] = {0,-1,1,-2,2,-2,2,-1,1};
const int dy[] = {0,-2,-2,-1,-1,1,1,2,2};

int main() {
  int n, m, hx, hy;
  cin >> n >> m >> hx >> hy;

  bool block[25][25] = {};           // 标记控制点
  for (int k = 0; k < 9; k++) {
    int x = hx + dx[k], y = hy + dy[k];
    if (x >= 0 && x <= n && y >= 0 && y <= m)
      block[x][y] = true;
  }

  ll dp[25][25] = {};
  dp[0][0] = 1;                         // ④ 边界

  for (int i = 0; i <= n; i++) {
    for (int j = 0; j <= m; j++) {
      if (block[i][j]) { dp[i][j] = 0; continue; }
      if (i == 0 && j == 0) continue;
      if (i > 0) dp[i][j] += dp[i-1][j]; // ③ 从上方来
      if (j > 0) dp[i][j] += dp[i][j-1]; // ③ 从左方来
    }
  }

  cout << dp[n][m] << '\n';             // ⑤ 答案
  return 0;
}
